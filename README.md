# InterfaceCreator

a simple utility to create an interface based on the source module

# Notes
* achieved using `ast` and `autopep8`(for formatting/ optional)

# How To Use
click shell help
```
infCreate --help
```

common usage
```
infCreate ./source.py ./target.py
```
